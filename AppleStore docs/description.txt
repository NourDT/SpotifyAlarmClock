Alarm Clock for Spotify is an Alarm Clock app, which allows you to wake up with your favorite Spotify songs. It supports multiple alarms and allows you to compile a playlist for your alarm.

To use the Spotify functionality of this application, a Spotify Premium account is mandatory.

Features

- LCD Clock 
- Support for multiple alarms
- Compile playlists for alarms
- Shuffle playlist
- Select days to repeat alarm
- Snooze functionality
